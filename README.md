# algorithms-2020

## Установка

*Для установки зависимостей необходима среда [nodejs](https://nodejs.org/en/)*

```bash
$ cd ./tests-runner
$ npm install
```

## Запуск тест раннера

```bash
$ node ./tests-runner ./01-lecture/1.Tickets/problem-02.js
```

## Запуск `*.test.js` файлов

```bash
$ npm test
```

## Сборка `.ts` файлов

```bash
$ npm run tsc
```
