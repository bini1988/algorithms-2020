## tests-runner

Cистема проверки на основе тестовых файлов

## Установка

*Для установки зависимостей необходима среда [nodejs](https://nodejs.org/en/)*

```bash
$ npm install
```

## Использование

```js
// problem.js
module.exports = function (lines) {
  // Processing and return result
  return true;
};
```

```bash
$ node tests-runner problem.js
```
